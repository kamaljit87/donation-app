# Add Srila Prabhupada Image

To display the image on the About page, save it to:

```
/var/www/donation-app/frontend/public/images/srila-prabhupada.jpg
```

## Quick Command:

```bash
# Upload your image file, then rename it:
mv /path/to/your/image.jpg /var/www/donation-app/frontend/public/images/srila-prabhupada.jpg
```

The page is already configured to display this image. If the image is not found, it will show a placeholder automatically.
